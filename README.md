# 🏏 StumpTalk – Cricket Blog & Quiz App

StumpTalk is a cricket-themed web application built using Node.js, Express.js, EJS, and React.  
It allows users to create blog posts and also play an interactive cricket quiz.

---

## 🚀 Features

### 📝 Blog System (Backend - Node + Express)
- Create, edit, and delete posts
- View all blog posts
- Cricket-themed UI
- Login system (basic authentication)

### 🔍 Explore Page
- Search posts by title
- Filter posts by category
- Built using JavaScript DOM manipulation

### 🎮 React Quiz App
- Cricket trivia quiz
- Score tracking using React hooks
- Class component used for scoreboard

---
